Mi proyecto consiste en una pagina web de contrataciones, tanto para empleados que quiieren buscar trabajo como para empleadores en publicar su perfil y sus ofertas.

modelos:
	empleos--- atributos: Cargo; detalles; ubicacion; sueldo.
	empleados	"   : nombre; apellido; presentacion; experiencia; estudios.    
	empleador	"   : nombre; apellido; profesion; edad; nombreEmpresa.

funcionalidad: La apliacion permite cargar datos en la base de datos tanto de empleos como el curriculum de los empleados y las especificaciones de los empleadores, a travez
de forms especificos para cada modelo. Tambien se puede buscar datos por filtrado en la lista de cada modelo y ampliar los datos en cada pagina.
